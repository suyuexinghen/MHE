# QCompute Tested Support Matrix

This matrix reflects current repository evidence for the QCompute extension. Status values use the following meanings:

- `tested` — automated tests or directly reviewed runnable examples cover the feature today
- `experimental` — implementation exists and has some evidence, but coverage or execution scope remains limited
- `gated` — supported only behind explicit environment/dependency gates such as optional packages or hardware tokens

| Family | Contract / Compiler Surface | Runtime / Evidence Coverage | Diagnostics / Policy / Governance Coverage | Test Anchor | Caveat / Not Yet Proven | Status |
